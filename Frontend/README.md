# AI Dashboard Frontend

React frontend byggd med Vite och Tailwind CSS. Projektet är strukturerat med återanvändbara komponenter och kräver Node.js 22 eller senare.

## Kommandon

```bash
npm install
npm run dev
npm run build
npm run lint
```

Devservern kör som standard på `http://127.0.0.1:5173/`.

## Lokala vyer

- Publik startsida: `http://127.0.0.1:5173/`
- Kundvy utan inloggningskrav: `http://127.0.0.1:5173/app`
